Näed ainult adminina: mängijate lisamine, mängude laadimine, tulemuste ja algusaegade testimine. Tulemusi saab sünkroniseerida ka API-Football tasuta plaaniga ning käsitsi muutmine jääb alles.
Admin juhend
API tervis kontrollib, kas server vastab.
Lae ametlik mängutabel uuendab FIFA 2026 mängud andmebaasis.
Sünkroniseeri ajakava uuendab mänguaegu olemasoleva ajakava järgi.
Sünkroniseeri tulemused tõmbab API-Footballist lõppenud mängude skoorid.
Käsitsi tulemus sisesta mängu numbri järgi, et testida punktiarvestust.
Käsitsi mänguaeg sisesta kuupäev kujul DD:MM:YY ja kellaaeg HH:MM Eesti aja järgi, et testida 1 tunni lukku.
Kustuta mäng vali mäng dropdown-listist. See kustutab ka selle mängu ennustused.
Lisa uus mäng sisesta mängu nr, meeskonnad, kuupäev DD:MM:YY, kellaaeg HH:MM ja asukoht. Etappi eraldi sisestama ei pea.
Import ajad (FIFA ajakava) on admini abivahend, kuhu saab kleepida mitu mänguaega korraga kujul mängu_nr,YYYY-MM-DD,HH:MM. See uuendab olemasolevate mängude algusaegu massina, kui FIFA ajakavas on vaja kellaaegu korraga parandada.
